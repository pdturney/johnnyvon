Self-Replicating Automata in Two-Dimensional Continuous Space

Continuous Von Neumann Machines

Requirements

The program requires Java 2 Runtime Environment (JRE), version 1.3 or above.

Launching the Java Applet

Launch Java applet specified with OBJECT tag
• understood by Internet Explorer

Launch Java applet specified with EMBED tag
• understood by Netscape

Launch Java applet specified with APPLET tag
• uses the browser's default Java Virtual Machine (JVM) to run the applet
• please make sure that the JVM understands the Java 2 API.
